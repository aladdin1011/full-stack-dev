1.Машинный код (1940-е годы):
    Машинный код — это низкоуровневый язык, состоящий из двоичных инструкций, которые непосредственно понимаются процессором.
2.Ассемблер (1940-е годы):
    Ассемблер использует мнемоники вместо двоичных инструкций, что позволяет программистам работать с более понятными командами.
3.FORTRAN (1957):
    FORTRAN (Formula Translation) считается первым высокоуровневым языком программирования, созданным для научных и инженерных расчетов.
4.COBOL (1959):
    COBOL (Common Business-Oriented Language) разработан для бизнес-приложений и до сих пор используется в банковской и финансовой сферах.
5.C (1972):
    Язык C оказал значительное влияние на многие современные языки программирования, такие как C++, Java и Python.
6.Java (1995):
    Java была разработана с акцентом на переносимость и безопасность, с лозунгом "Write Once, Run Anywhere".
7.Python (1991):
    Python был разработан с акцентом на простоту и читаемость кода, что сделало его популярным в научных и веб-разработках.
8.JavaScript (1995):
    JavaScript был создан за 10 дней и стал основным языком для разработки веб-приложений и взаимодействия с пользователем на клиентской стороне.
